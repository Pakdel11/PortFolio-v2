About

An online booking system is very important to everyone. By having a system that can find free time and book it immediately in a minute is a big improvement in our modern life. In this project I want to work on this system and find solutions that can be useful to people. It would be very interesting to build such an application. It can be a web application or an app on cell phone devices.

Metuder som har använt för färghjulet

Jag använt min egen metod för att skilja mellan kombination av olika färger. Jag gör en fyra olika boxar som innehåller de färger som jag tycker att dem ska vara bakgrundsfärger och gör olika former och texter med olika färger för att se vilka av dessa färger passar bäst. Tillslut tar jag beslut från dessa fyra boxar att välja en av dessa. För detta projekt valde jag vita bakgrund boxen.

Method

For the method I am going to use the MERN stack method. MERN stands for MongoDb which is the database side, Express JS is a back-end web framework and React is one of the Javascript libraries that is a front-end framework and finally last but not least Node.js which is for the server side a back-end runtime environment. I can use these resources to answer my first question which is how to build an online booking system.
